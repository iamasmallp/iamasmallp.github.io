    介绍

    欢迎使CMex开发者文档。

    本文档概述了交易功能、市场行情和其他应用开发接口。

    API分为两部分：REST API和Websocket 实时数据流

    REST API包含四个类别：用户模块（私有），交易模块（私有），市场数据（公共），杠杆交易，其他接口（公共） Websocket包含两类：公共频道和私人频道

    鉴权

    创建api key

    通过接口进行请求前，您需在联系客服创建API-KEY。创建成功后，您需妥善保管好以下两条信息： * access_key * secret_key access_key和secret_key由系统随机生成并提供。以上信息若遗失将无法恢复，需要重新申请api key。

    API权限

    API权限分为以下几类：

    • 通用权限 - 允许API访问大部分的GET请求。
    • 交易权限 - 允许API具有下单权限。
    • 提现权限 - 计划支持。

    创建请求

    Rest请求头必须包含以下内容:

    • version 签名版本，默认值1.0
    • access-key access_key以字符串传递
    • algorithm 签名算法，默认值HmacSHA256
    • signature 签名
    • timestamp 请求的时间戳，当前时间10分钟之内有效

    使用 API-Secret 对 http method + version + algorithm + timestamp + url path + queryString + body 拼接的字符串进行HMAC-sha256加密,再将加密内容使用 base64 编码。

    • http method (GET，POST，DELETE 大写)
    • url path /api/v1/timestamp
    • queryString ?symbol=BTC-USDT
    import base64
import hashlib
import hmac
import time
import json
import requests

class BaseRequest(object):

    def __init__(self, url='', version='1.0', algorithm='HmacSHA256', access_key='', secret_key=''):
        self.url = url
        self.version = version
        self.algorithm = algorithm
        self.access_key = access_key
        self.secret_key = secret_key

    def request(self, method, uri, params=None, body=None):
        query_string = ''
        body_string = ''
        if body:
            body_string = json.dumps(body)
        now = int(time.time())
        if params:
            for k, v in params:
                query_string += "{}={}".format(k, v)
        str_to_sign = method + self.version + self.algorithm + str(now) + uri + query_string + body_string
        sign = base64.b64encode(
            hmac.new(self.secret_key.encode('utf-8'), str_to_sign.encode('utf-8'), hashlib.sha256).digest())
        headers = {
            "SIGNATURE": sign,
            "TIMESTAMP": str(now),
            "ACCESS-KEY": self.access_key,
            "ALGORITHM": self.algorithm,
            "Content-Type": "application/json"
        }
        return requests.request(method, self.url + uri, headers=headers, data=body_string)

    exchange/private/channel/users/{userNo}

    请求参数

    参数名称 数据类型 是否必须 默认值 描述
    userNo long true NA 用户编号

    响应参数

    字段名称 数据类型 描述
    topic string 业务类型:ORDER,ACCOUNT
    userNo long 用户编号
    data object 业务实体数据

    data参数说明:

    • topic为ORDER时
    参数名称 数据类型 描述
    event string 事件类型:DEAL,CANCEL
    orderNo long 订单号
    symbol string 交易对
    price string 限价单交易价格
    size string 数量
    fillSize string 订单中已成交部分的数量
    fillFees string 已交交易手续费总额
    priceAverage string 成交均价
    side string 方向:buy(买),sell(卖)
    type string 订单类型:lmt(限价单); mkt(市价单)
    status string 订单状态，包括created, partial-filled,filled,partial_canceled,canceled
    createTime long 时间戳
    • topic为ACCOUNT时
    字段名称 数据类型 描述
    event string 事件类型:SETTLE
    accountType string 账户类型:spot_account(现货),assets_account(资产)
    currency string 币种
    available string 可用额度
    total string 总额度
    frozen string 冻结额度
